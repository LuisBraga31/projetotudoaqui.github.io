HTML

HTML (Linguagem de Marcação de HiperTexto) é o bloco de construção mais básico da web. Define o significado e a estrutura do conteúdo da web. Outras tecnologias além do HTML geralmente são usadas para descrever a aparência/apresentação (CSS) ou a funcionalidade/comportamento (JavaScript) de uma página da web.

Dicas

Tag Descrição
h1 ... h6 Variações para escrita de cabeçalho.
div Utilizado para dividir os conteúdos de uma página em blocos.
p Texto sem formatação, geralmente paragráfos.
hr Pula uma linha e desenha uma barra horizontal indicando fim da sessão.
small Texto em tamanho reduzido.
a Tag de âncora, utilizada para incluir hiperlinks.
img Tag para mostrar imagens.
ol Tag para criar uma lista ordenada ou numerada de itens.
ul Tag para criar uma lista não ordenada de itens.
form Tag para criar um formulário.
button Tag para criar um botão na página.
table Tag para criar uma tabela como essa.
object Tag para criar um tipo de mídia, podendo ser aúdio, vídeo, documento pdf, etc.
figure Tag para criar figuras como diagramas e gráficos.
nav Tag para criar itens de navegação.
